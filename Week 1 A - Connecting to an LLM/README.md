# Week 1 A - Connecting to an LLM

## Objective

Learn how applications communicate with LLMs and turn that connection into small useful projects.

## Projects

1. **Calling an LLM API** — Send one prompt to an LLM from Python.
2. **Website Scraper and Summarizer** — Turn a website URL into an AI summary.
3. **LLM Arena Battle** — Compare two anonymous model responses in a Gradio app.

## Requirements

Each project uses Python 3.11+, `uv`, and a Groq API key. Every project has its own `.env.example`, `pyproject.toml`, and `uv.lock`.

## How to run

Open the project you want to run, then install its dependencies:

```powershell
uv sync
```

Use the project README for its exact run command and required `.env` variables.
