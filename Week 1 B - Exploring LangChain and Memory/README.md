# Week 1 B - Exploring LangChain and Memory

## Objective

Learn how LangChain helps organize LLM applications, how chat memory works, and how an LLM can request local Python tools.
 

## Projects

1. **Website Summarizer with LangChain** — Rebuild the website summarizer as a LangChain pipeline.
2. **Chatbot with Memory** — Show fixed history and growing in-memory chat history.
3. **Tool-Using Shop Assistant** — Let an LLM request a local price lookup tool through a Gradio chatbot.

## Requirements

Each project uses Python 3.11+, `uv`, and a Groq API key. Every project has its own `.env.example`, `pyproject.toml`, and `uv.lock`.

## How to run

Open the project you want to run, then install its dependencies:

```powershell
uv sync
```

Use the project README for its exact run command and required `.env` variables.
